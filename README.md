# roondisplay
Display using Raspberry Pi with Roon API

Fetches the playing track and artist via the Roon API on the local network.


Parts:
Raspberry Pi Zero
[SSD1309 OLED Display](https://www.aliexpress.com/item/1005003097088323.html)
